# PrivacyGuard

Lean landing site for PrivacyGuard, a consumer privacy guidance product focused on California DROP and broader privacy-deletion hygiene.

## Scope
- Explain California DROP in plain language using official California sources
- Distinguish free official tools from paid guidance
- Offer a single $9/month subscription via Stripe Checkout
- Deploy as a static site on Vercel

## Notes
This version is intentionally simple and does not claim to submit DROP requests on a user's behalf.
